# LaTeX Expert Reference Index

Source: extracted from [`latex_syntax.md`](../../latex_syntax.md), based on the January 2025 LaTeX2e unofficial reference manual.

Use these files for core LaTeX2e syntax, structure, environments, math, and macro authoring.
This reference is strongest for base LaTeX and the standard material covered in the source manual.
For package-specific behavior outside that scope, inspect the package documentation separately.

## Search first

- Use `rg -n 'pattern' latex-expert/references` to find the right file before opening a large reference.
- Start with the chapter number when the topic is obvious, such as math (`16-...`) or environments (`08-...`).
